# Assess Command

Evaluate your current knowledge and skill level. This command provides a personalized assessment helping you understand your strengths, identify gaps, and get customized learning recommendations.

## Usage

```
/assess
```

## Assessment Areas

### 1. **Language & Syntax Proficiency**

**Programming Languages Covered**:
- JavaScript/TypeScript
- Python
- Java
- Go
- Rust
- PHP
- Kotlin
- C++

**Assess**: Can you write and understand code in your preferred language?

**Beginner Level**:
- Know basic syntax (variables, functions, loops)
- Can run "Hello World"
- Understand simple programs

**Intermediate Level**:
- Can build small applications
- Understand OOP concepts
- Can debug issues independently

**Advanced Level**:
- Design elegant solutions
- Optimize for performance
- Mentor others in the language
