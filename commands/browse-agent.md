# Browse Agent Command

Explore all 7 specialized agents in the Developer Roadmap plugin. Each agent covers a specific domain with detailed guidance, resources, and learning paths.

## Usage

```
/browse-agent
```

## How to Choose an Agent

1. **By Career Interest**: Choose the agent matching your desired role
2. **By Experience Level**: Start with fundamentals, advance gradually
3. **By Current Skills**: Pick an agent adjacent to your expertise
4. **By Project Needs**: Choose based on immediate project requirements
5. **By Learning Style**: Some agents focus on theory, others on practice

## Multi-Agent Learning

**Recommended Learning Combinations**:
- **Full Stack**: Frontend + Backend + DevOps
- **Mobile Full Stack**: Mobile + Backend + DevOps
- **Data-Driven**: Data/AI + Architecture + Backend
- **Product Focus**: Architecture + UX Design + Product Management
- **Security-Focused**: Security + Backend + DevOps
